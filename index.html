¿Que es HTML?

      HTML (HyperText Markup Language) es un lenguaje de marcado que se utiliza para estructurar y desplegar páginas web. Es el componente básico de la web y define el significado y la estructura del contenido de una página web.

Historia HTML

      El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, investigador del CERN (Organización Europea para la Investigación Nuclear) propuso un nuevo sistema de “hipertexto” para compartir documentos. Los sistemas de hipertexto habían sido desarrollados años antes.

    • Tu Primer Documento HTML

        Para crear un primer documento HTML, puedes seguir los siguientes pasos:

    • Abrir un procesador de textos, como el Bloc de Notas de Windows o el Procesador de textos de Ubuntu.
    • Escribir las etiquetas HTML en el editor de texto
    • Guardar el documento con la extensión *.HTML o *.HTM
    • Abrir el documento con doble clic en el icono de HTML

    Que son Etiquetas en HTML

        Las "tags" HTML, o "etiquetas" HTML, son códigos utilizados para "marcar" el texto de una página web, con el fin de dar instrucciones al navegador sobre cómo mostrarlo.

    Que son Atributos en HTML

        Los atributos en HTML son fragmentos de código que se utilizan para modificar el comportamiento o la visualización de los elementos HTML. Se agregan a las etiquetas HTML utilizando la sintaxis "nombre de atributo=valor".

    Reglas Fundamentales en HTML

  • Las etiquetas HTML se usan para designar los elementos de una página web.
  • Las etiquetas de apertura comienzan con el símbolo < y las de cierre con el símbolo
  • La mayoría de los elementos HTML tienen una etiqueta de apertura y otra de cierre.
  • Los nombres de los elementos no distinguen entre mayúsculas y minúsculas.
