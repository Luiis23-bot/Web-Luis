Atributos Globales:

Los atributos globales son atributos comunes a todos los elementos HTML; se pueden usar en todos los elementos, aunque es posible que no tengan ningún efecto en algunos elementos. Los atributos globales se pueden especificar en todos los elementos HTML, incluso aquellos no especificados en el estándar.

Atributos Form:

El elemento HTML form representa una sección de un documento que contiene controles interactivos que permiten a un usuario enviar información a un servidor web. Es posible usar las pseudo-clasess de CSS :valid e :invalid para darle estilos a un elemento form.

Atributos Input:

La etiqueta en HTML es un elemento importante para cualquier desarrollador web debido a su versatilidad y su capacidad para recopilar información del usuario, ¡por eso es una herramienta fundamental en la creación de formularios web interactivos!.
