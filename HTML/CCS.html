CSS

1-CSS (Cascading Style Sheets) es un lenguaje de programación que permite controlar el diseño y presentación de las páginas web. Se utiliza para dar estilo a los elementos HTML, como el color, la fuente, la disposición, las animaciones y los efectos visuales. CSS es un lenguaje basado en reglas, donde cada regla se abre con un selector que identifica el elemento HTML al que se le aplicará el estilo. Por ejemplo, la regla h1 { color: red; font-size: 5em; } especifica que el encabezado principal de una página se muestre en rojo y en letras grandes.

Como Utilizar CSS Con Html

2-

  • Seleccionar un elemento en el documento HTML.
  • Abrir el menú contextual del navegador y seleccionar "Editar CSS".
  • En la ventana de CSS, escribir la regla CSS que desea aplicar al elemento HTML seleccionado.
  • Guardar el documento HTML.
  • Ver el resultado en el navegador.

    • CSS en linea

    3-El CSS es un lenguaje basado en reglas: cada usuario define las reglas que especifican los grupos de estilos que van a aplicarse a elementos particulares o grupos de elementos de la página web. Por ejemplo: «Quiero que el encabezado principal de mi página se muestre en letras grandes de color rojo».

    CSS interno

    4-CSS interno es un estilo de CSS que se aplica directamente a los elementos HTML. CSS interno se aplica a los elementos HTML directamente, sin necesidad de un archivo externo.

    CSS externo

    5-CSS externo es un estilo de CSS que se aplica a los elementos HTML mediante la inserción de un archivo externo.
